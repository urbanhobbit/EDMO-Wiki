# Code of Practice on Disinformation (Uygulama Kodu)

Büyük çevrimiçi platformların dezenformasyonla mücadelede gönüllü olarak benimsediği AB çapındaki davranış kodu. EDMO bu kodun uygulanmasını izleyen ve değerlendiren temel kurumlardan biri.

**Kronoloji:**
- **Haziran 2022:** 34 imzacıyla "Güçlendirilmiş Uygulama Kodu" (Strengthened Code of Practice) yürürlüğe girdi. EDMO kalıcı görev gücünde yer aldı.
- **2023:** EDMO Yönetim Kurulu, imzacı platformları şeffaflık ve raporlama doğruluğu eksikliği nedeniyle sert biçimde eleştirdi. Twitter'ın (X) API ücretlendirmesi özel olarak vurgulandı.
- **2024:** [[entities/gadmo|GADMO]] CoP Monitor 2024 Raporu: platform önlemlerinin seçim dönemlerinde yetersiz kaldığını, deepfake ve yasadışı siyasi reklamcılık vakalarıyla belgeledi.
- **2025:** "Dezenformasyon Uygulama Kodu" (CoPD) yeniden adlandırılarak "Davranış Kodu" (Code of Conduct) haline geldi.

**VLOPSE Uyum Değerlendirmesi (2025):**
EDMO'nun büyük platformlara yönelik uyum değerlendirmesi üç temel eksiklik tespit etti:
1. Şeffaflık yetersizlikleri
2. Bağımsız denetim boşlukları
3. Yetersiz sonuç metrikleri
